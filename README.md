# Dashboards_Temp

Traffic Light DB (Power BI)
- Traffic light KPI indicator inside a table which some people seem to like
- Conditionally format the KPI column font colour (green, yellow, red) based on another column (like sales or profit)
- Use UNICHAR(11044) which is the unicode representation for a (black) circle icon 
- Link: https://www.alt-codes.net/circle-symbols
- Dataset Used: Sample Superstore
